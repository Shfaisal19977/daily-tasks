# Introduction

A Laravel-based REST API for managing projects, tasks, comments, books, and products.

<aside>
    <strong>Base URL</strong>: <code>http://laravel1.test</code>
</aside>

    Welcome to the Project Management System API documentation. This API provides comprehensive endpoints for managing projects, tasks, comments, books, and products.

    ## Getting Started

    All API requests should be made to the base URL: `http://laravel1.test/api`

    The API uses standard HTTP methods (GET, POST, PUT, PATCH, DELETE) and returns JSON responses.

    ## Features

    - **Projects**: Create and manage projects with details like name, description, dates, and status
    - **Tasks**: Create and manage tasks within projects with priority, status, and due dates
    - **Comments**: Add comments to tasks for collaboration and tracking
    - **Books**: Manage a library of books with title, author, and publication year
    - **Products**: Manage products with inventory tracking and stock reduction capabilities
